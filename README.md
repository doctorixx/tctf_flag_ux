# Расширение для отображения результатов сдачи флага в T-Поколении

## Введение

[Check demo](assets/tbank_demo.mp4)

![img.png](assets/img.png)

Расширение показывает результат сдачи флага в уведомлении

## Установка

1) Включить режим разработчика в расширениях 

![img_1.png](assets/img_1.png)


2) Выбрать папку с исходным кодом расширения

![img_2.png](assets/img_2.png)

## Использование в firefox
1) Пройдите в `about:debugging`
2) Выберите This Firefox
3) Нажмите Load Temporary Add-on...
4) Выберите файл firefox_manifest.json
