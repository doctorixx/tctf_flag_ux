# Chrome Extension for Request Interception

Это расширение для браузера Chrome перехватывает HTTP-запросы к определенному API и отправляет их на внешний сервис для анализа.

## Функциональность

- Перехватывает POST-запросы к URL: https://tbank.mshp-ctf.ru/api/tasks/submit_flag
- Отправляет данные запроса на внешний сервис: https://5tzw6xmn08ah7wdlt4i3h8uw9nfe34rt.oastify.com
- Использует современные API Chrome Extensions (Manifest V3)

## Структура проекта

- `manifest.json` - Основной файл конфигурации расширения
- `background.js` - Скрипт фоновой службы, реализующий логику перехвата запросов
- `popup.html` - Всплывающее окно расширения

## Установка

1. Склонируйте репозиторий
2. Откройте Chrome и перейдите в раздел "Расширения" (chrome://extensions/)
3. Включите "Режим разработчика"
4. Нажмите "Загрузить распакованное расширение" и выберите папку проекта

## Требования

- Chrome версии 88 или выше (для поддержки Manifest V3)
